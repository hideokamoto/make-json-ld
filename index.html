About

カスタムフィールドのデータをJSON-LD形式で出力するプラグインです。

カスタムフィールド名を「schema:name」のような形式にすることで、簡単にJSON-LD化されたデータをみることができます。

仕様

  • 管理画面から登録できる語彙は１つのみ
  • 語彙を登録した場合、その語彙しか使えない
  • 管理画面から登録された語彙が無い場合、デフォルトの語彙が出力される
  • /jsonld-context/に表示されるURLがエンコードされて表示されるが動作に問題はない
  • 語彙１つだけの時は「:(CompactIRIs)」を使わずに定義する。
    • 語彙１つの時→「 "name" : "YourName"」
    • 語彙複数の時→「"schema:name" : "YourName"」

デフォルトで使用できる語彙

  • "rdf" : "http://www.w3.org/1999/02/22-rdf-syntax-ns#",
  • "rdfs" : "http://www.w3.org/2000/01/rdf-schema#",
  • "vCard" : "http://www.w3.org/2006/vcard/ns#",
  • "foaf" : "http://xmlns.com/foaf/0.1/",
  • "dc" : "http://purl.org/dc/elements/1.1/",
  • "dcterms": "http://purl.org/dc/terms/",
  • "cal" : "http://www.w3.org/2002/12/cal/icaltzd#",
  • "geo" : "http://www.w3.org/2003/01/geo/wgs84_pos#",
  • "owl" : "ttp://www.w3.org/2002/07/owl#",
  • "schema" : "http://schema.org/",
  • "skos" : "http://www.w3.org/2004/02/skos/core#",
  • "yafjp" : "http://fp.yafjp.org/terms/place#"

*それぞれの語彙については(http://fp.yafjp.org/yokohama_art_lod/place_rdf)[http://fp.yafjp.org/yokohama_art_lod/place_rdf]を参照してください。
